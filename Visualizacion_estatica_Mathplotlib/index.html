1. Gráfico de Barras (Bar Chart)

Uso Correcto

"Se recomiendan para comparar categorías; funcionan mejor cuando el eje de magnitud es claro y las categorías se ordenan cuando el objetivo es ranking."

Wilke, 2019
python implementation
# Datos
categorias = ['A', 'B', 'C', 'D', 'E']
valores = [45, 82, 35, 95, 60]

# 1. ORDENAR (Clave para rankings)
df = pd.DataFrame({'cat': categorias, 'val': valores})
df = df.sort_values('val')

fig, ax = plt.subplots()

# 2. GENERAR (Barras horizontales para leer mejor las etiquetas)
bars = ax.barh(df['cat'], df['val'], color='#487eb0')

# 3. DESTACAR (Storytelling)
bars[-1].set_color('#e84118') # El mayor en rojo

ax.set_title("Ventas por Producto (Ranking)")
ax.spines['bottom'].set_visible(False) # Limpieza
ax.xaxis.set_ticks([]) # Menos tinta

2. Gráfico de Líneas (Line Chart)

Uso Correcto

"Se usan para tendencias, secuencias o series temporales; la lectura se favorece cuando el eje temporal es consistente y el diseño evita saturación."

3. Dispersión (Scatter Plot)

Uso Correcto

"Se utiliza para relación entre dos variables y para detectar patrones, agrupamientos u outliers; es especialmente útil en exploración."

python implementation
fig, ax = plt.subplots()

# 1. DISPERSIÓN
# 'alpha' ayuda a ver solapamientos (clusters)
ax.scatter(x, y, alpha=0.6, color='blue')

# 2. ANOTACIÓN DE OUTLIERS
# Fundamental para análisis exploratorio
ax.annotate('Outlier', xy=(8, 5), xytext=(5, 10),
            arrowprops=dict(facecolor='black', arrowstyle='->'))

ax.set_title("Inversión vs Retorno")
ax.set_xlabel("Inversión")
ax.set_ylabel("ROI")

5. Diagrama de Caja (Boxplot)

Uso Correcto

"Sintetizan mediana, variabilidad y posibles outliers; son útiles para comparar distribuciones por grupos sin recargar el gráfico."

Few, 2012; Wilke, 2019
python implementation
data = [grupo1, grupo2, grupo3]

fig, ax = plt.subplots()

# 1. CREACIÓN
# patch_artist=True permite colorear las cajas
cajas = ax.boxplot(data, patch_artist=True, 
                   labels=['G1', 'G2', 'G3'])

# 2. ESTILIZACIÓN DE GRUPOS
colors = ['#dfe6e9', '#74b9ff', '#a29bfe']
for patch, color in zip(cajas['boxes'], colors):
    patch.set_facecolor(color)

ax.set_title("Comparación de Grupos")
ax.set_ylabel("Valor")
